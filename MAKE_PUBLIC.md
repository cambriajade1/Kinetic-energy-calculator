## Option 1: GitHub Pages (Easiest - Free Hosting!)

### Steps to Enable GitHub Pages:

1. **Go to Your Repository**
   - Visit: https://github.com/cambriajade1/Kinetic-energy-calculator

2. **Open Settings**
   - Click the **Settings** tab at the top right

3. **Find GitHub Pages**
   - In the left sidebar, scroll down and click **Pages**

4. **Configure GitHub Pages**
   - Under "Source", select **Deploy from a branch**
   - Under "Branch", select **main**
   - Choose folder: **/root** (or just use the default)
   - Click **Save**

5. **Wait a Moment**
   - GitHub will process your deployment (usually takes 1-2 minutes)

### Your Public URL:

https://cambriajade1.github.io/Kinetic-energy-calculator/

## Option 2: Vercel (Even Better - Automatic Deployments)

1. Go to https://vercel.com and sign up
2. Connect your GitHub account
3. Import your repository
4. Vercel will auto-deploy your site
5. Get a live URL instantly

---

## Option 3: Netlify (Another Great Option)

1. Go to https://netlify.com and sign up
2. Click **New site from Git**
3. Select your GitHub repository
4. Netlify will deploy automatically
5. You'll get a unique URL
